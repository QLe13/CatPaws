Project Description

The objective of this project is to develop a user-friendly and efficient software solution for class registration in a higher education environment. Our focus is to deliver a lightweight and reliable service that fulfills our clientâ€™s specific requirements without adding unnecessary features that can complicate the user experience. We will draw inspiration from existing registration services such as TigerPaws from Trinity University and Skyward from Texas public schools. However, we will avoid common issues identified in these applications, such as unintuitive interfaces, feature bloat, and frustrating quirks that slow down or confuse users. Our main goal is to create a straightforward software that simplifies the registration process and helps students navigate through their course selection with ease. We aim to reduce stress and frustration during the hectic time of class registration by providing a seamless and intuitive user experience.

Functional Project Requirements

Student Registration: The system should allow students to register for classes, view their class schedules, and drop or add courses.

Faculty Registration: The system should allow faculty members to view their schedules, access/add/delete course information.

Course Management: The system should allow faculty to add classes and manage course details such as course information, location, schedule.

Non-Functional Project Requirements

Security: The system should be secure, with authentication and authorization measures in place to prevent unauthorized access and protect sensitive data.

Performance: The system should be able to handle a large number of simultaneous users without performance degradation.

Reliability: The system should be reliable and available 24/7, with a minimum of downtime for maintenance.

Usability: The system should be easy to use and intuitive, with clear instructions and help resources available for users.

Scalability: The system should be scalable, able to handle a growing number of users and courses.
